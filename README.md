## Job Tracker Chrome Extension
A lightweight Chrome extension that helps users track job applications directly from their browser. It allows you to save job postings, record application status, add notes, and organize opportunities in one place without switching tools. Designed to reduce tab overload and manual tracking, the extension streamlines the job search workflow and keeps everything accessible while browsing job boards.
